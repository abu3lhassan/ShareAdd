# ShareAdd

**ShareAdd** is a personal Android app for saving, organizing, and sharing places and addresses in a clean and practical way.

## Features

- Save places and addresses
- Saved Locations page
- Premium dashboard
- Categories
- Place images
- Notes
- Star rating
- Favorites and pinned places
- Search and filtering
- Arabic and English support
- Dark mode and light mode
- PIN / biometric lock
- Export and import backup files
- Import safety preview
- Share places with clean text templates

## Backup and Import

ShareAdd uses Android's file picker for backup and restore.

You can export a backup file to your phone, Downloads folder, or Google Drive.  
You can also import a backup file from your phone or Google Drive.

No Google Drive API, sync, login, or cloud setup is required.
